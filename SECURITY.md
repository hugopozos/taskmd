# Security Policy

## Supported versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a vulnerability

TaskMD is a local-first tool that binds to `127.0.0.1` by default. If you
discover a security issue, please open a private issue on GitHub or contact the
maintainer directly.

Do not report security issues via public issues or discussions.
